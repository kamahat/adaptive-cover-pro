# v2.20.1

## 🔑 Highlights

Two follow-up fixes for the venetian dual-axis sequencer that shipped in v2.20.0. The first hardens the post-close handoff against actuators (e.g. FGR223) that reassert their tilt cache shortly after the carriage stops. The second closes a gap in the tilt-only update path that bypassed the `tilt_skip_above` guard, causing redundant tilt commands to fire on every solar recompute while the carriage sat at or above the threshold.

## 🐛 Bug Fixes

- **Post-settle tilt command races actuator firmware tilt-reassert (#33):**
  Some actuators (e.g. FGR223) reapply their internal tilt cache shortly after the
  carriage finishes closing. When the sequencer sent `set_cover_tilt_position`
  immediately after detecting settle, the command could land mid-reassert and be
  overwritten by the actuator. The root cause was two separate gaps:
  `_wait_for_position_settle` declared success on position-in-tolerance alone,
  without confirming the cover had left `opening`/`closing` state; and there was
  no hold after settle before the tilt command fired. The fix requires the cover
  state to be neither opening nor closing before settle is declared (when a
  `get_state` callback is provided), then holds `VENETIAN_POST_SETTLE_HOLD_SECONDS`
  (2.0 s) before dispatching `set_cover_tilt_position`. The no-callback path is
  preserved for backwards compatibility.

- **Tilt-only mode ignores `tilt_skip_above` threshold, flooding tilt commands (#33):**
  When the carriage was at or above the `tilt_skip_above` threshold (default 95%),
  `maybe_update_tilt_only` bypassed the retraction guard that
  `after_position_command` already enforced. Every solar recompute fired a redundant
  `set_cover_tilt_position` call — one user observed 31 tilt commands in a single
  pre-sunset hour on v2.20.0. The fix extracts the threshold rule into a shared
  `_resolve_skip_above_tilt` helper used by both call sites, so the guard is applied
  consistently regardless of which sequencer path is active. No behavior change when
  the carriage is below the threshold.
