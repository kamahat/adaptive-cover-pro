## Highlights

2026.7.4 is a narrow follow-up to last release's venetian Tilt Drift Reset fix (#927). That fix stopped a late ENDPOINT publish from tripping a false manual override by value-matching it against a one-shot stamp, but #930 reported that the value-match left two paths open: a mid-excursion INTERMEDIATE publish landing between the target and the endpoint, and a return-leg publish arriving after the endpoint publish had already burned the stamp. Both still scored as manual moves. No configuration changes and nothing to migrate.

## Fixed

- **Venetian Tilt Drift Reset no longer trips on mid-excursion or return-leg tilt publishes (#930):** The #927 fix shipped in 2026.7.3 recognized a late ENDPOINT publish by matching its wire value against a one-shot `_ResetExcursion` stamp and popping it on the first match. That still left two gaps: a mid-excursion INTERMEDIATE publish — neither the endpoint nor the target, like the reporter's `28` reading at a 7% delta on the return leg — fell outside the endpoint-only match and scored as a manual move, and because the endpoint publish itself consumed the stamp, any later return-leg publish had no record left to protect it. Separately, the `wait_for_target` gate returned early without advancing the stamp at all — only the command-grace gate did. `_ResetExcursion` now carries the LOGICAL `target` alongside the endpoint plus an `endpoint_seen` flag, so `is_reset_excursion_publish()` tracks the whole out-and-back trajectory instead of a single value: an endpoint publish marks `endpoint_seen` and retains the record, a publish landing back on `target` consumes it, and — via the new `_publish_matches()` helper — any wire value inside the traversal band between the endpoint and target suppresses without consuming. A shared `_reject_gated_update()` helper now runs `consume_excursion()` from both the command-grace and `wait_for_target` gates, so the trajectory keeps advancing under either short-circuit. A user move to the mirror value, or a near-target value alongside a simultaneous genuine position move, still trips override as before.
